This is the backend server for News Feed App.
Setup Instructions--

1. npm install
2. npm start or npm run dev

======News Add api====

Endpoint==> http://localhost:3000/api/news/add
Method==> Post
FormData==> {
image:'',
headline='',
category='',
author='',
}

== All News List ====
Endpoint==> http://localhost:3000/api/news/list
Method==> Get
